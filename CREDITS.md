# Credits and Third-Party Software

StudyShield's application code and bundled companion-pack format are maintained in this repository. The Android build uses:

- [Android Open Source Project](https://source.android.com/) and [AndroidX](https://developer.android.com/jetpack/androidx) - Android platform APIs, Activity, Jetpack Compose, Material 3, Lifecycle, Saved State, Room, DataStore, Media3, and AndroidX Test. AndroidX libraries are generally Apache-2.0; inspect each artifact for its definitive license.
- [Kotlin](https://github.com/JetBrains/kotlin) and [kotlinx.coroutines](https://github.com/Kotlin/kotlinx.coroutines) - language tooling and coroutine runtime; Apache-2.0.
- [JUnit 4](https://github.com/junit-team/junit4) - local JVM test framework; Eclipse Public License 1.0.
- [JSON-java](https://github.com/stleary/JSON-java) - JVM-test JSON implementation used outside the Android runtime; public-domain dedication text applies as documented upstream.

Bundled companion artwork, audio, and manifests belong to this repository's companion packs and are not sourced from the AndroidX projects above.
